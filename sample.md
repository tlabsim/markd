# Welcome to Markd 🎉

A beautiful, feature-rich markdown viewer and editor for desktop.

## Features

- **Live Preview** - See your markdown rendered in real-time
- **Split View** - Edit and preview side by side
- **Dark Theme** - Easy on the eyes with a gorgeous dark mode
- **File Browser** - Navigate your markdown files with ease
- **Search** - Find text within your documents
- **Export** - Export to HTML format
- **Cross-Platform** - Works on Windows, Linux, and macOS

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

    print(fibonacci(i))
```
